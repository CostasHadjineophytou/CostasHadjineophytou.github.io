App Engineering Video Demo Website

Next.js / Prisma / PostgreSQL

Overview

View on GitHub Watch on YouTube

Led a team of 4 to create a comprehensive video demo website for showcasing UCL projects to clients. The platform features robust administrative capabilities and user management systems.

Key Features

  • Client account registration and management
  • Project proposal submission system
  • Administrative dashboard for content management
  • Project listing and categorisation
  • News and events management
  • Responsive design for all devices

Technical Details

The application was built using:


  • Next.js for the frontend and API routes
  • Prisma as the ORM for database operations
  • PostgreSQL for data storage
  • React framework for the frontend
