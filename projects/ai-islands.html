IBM-UCL IXN AI Islands Desktop Application

Python / FastAPI / C# / .NET MAUI

Overview

Watch on YouTube

Worked in a team of 4 to produce a versatile and unified AI model collection application providing users with access to online and offline AI models in GenAI, NLP, and Computer Vision.

Key Features

  • Integrated IBM's watsonx AI models and services
  • Individual model configurations and model chaining configurations
  • Integrated Retrieval-Augmented Generation (RAG) system and Chatbot for LLMs
  • Inferencing models and model-chains
  • Focus on accessibility and user experience

Technical Details

The application was built using:


  • Python and FastAPI for the backend services
  • C# and .NET MAUI for the cross-platform desktop application
  • IBM watsonx AI services integration
