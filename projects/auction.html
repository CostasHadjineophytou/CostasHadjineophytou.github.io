Auction Website

PHP / SQL / HTML / CSS

Overview

View on GitHub Watch on YouTube

Led a team of 4 to develop a full-featured auction website with comprehensive bidding functionality, user management, and administrative capabilities.

Key Features

  • User account management system
  • Real-time bidding functionality
  • Auction creation and management
  • Automated notification system for outbid alerts, auction win notifications and new auction alerts
  • Personalised recommendation system
  • User watchlists
  • Administrative dashboard

Technical Details

The application was built using:


  • PHP for backend logic
  • MySQL database for data storage
  • HTML/CSS for frontend design
  • JavaScript for interactive features
