Primality & Factorial Calculator

MIPS 32 Assembly Language

Overview

View on GitHub

Produced a MIPS 32 Assembly Language program that calculates the prime status of a number and the factorial of a number.


To run this program, you'll need to use the MARS (MIPS Assembler and Runtime Simulator) IDE, which can be downloaded here. Simply load the assembly file into MARS, assemble the program, and run it to interact with the calculator.

Key Features

  • Primality Calculation
    • Efficiently determines if a given number is prime using division-based checking
    • Handles special cases for 0, 1, and 2 following mathematical conventions
  • Factorial Calculation
    • Computes the factorial of any non-negative integer
    • Handles edge case of 0! correctly, returning 1
  • User Interface
    • Interactive menu system with clear options for each operation
    • Continuous operation until user chooses to exit

Technical Implementation

  • Implemented using MIPS 32-bit assembly language with syscalls for I/O operations
  • Utilises efficient register management and loop structures for calculations
  • Implements proper subroutine calling conventions with jump-and-link (jal) instructions
  • Includes comprehensive error handling for edge cases in both calculations
